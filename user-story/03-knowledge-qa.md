# P0-03 基于知识库提问并获得答案

## 用户故事

作为培训负责人，我希望直接用自然语言向 AI 提问，得到基于已上传培训文档的准确答案，并能看到答案来自哪份文档的哪个段落，方便核实。

## 场景描述

用户在聊天面板输入问题，Agent 调用 rag_search 工具检索相关文档片段，结合检索结果生成带引用角标的回答，用户可 hover 查看原文出处。

## 前置条件

- 工作区已上传至少一份状态为"已就绪"的文档
- 聊天面板正常加载

## 核心步骤

1. 用户在聊天输入框输入问题（如"消防演练的频率要求是什么？"）
2. 点击发送，消息显示在对话流中
3. Agent 调用 `rag_search` 工具检索相关片段（工具卡片在对话中可见）
4. Agent 综合检索结果生成回答，回答中包含引用角标（如 ⟦1⟧）
5. 用户 hover 引用角标，弹出浮层显示来源文档名和原文段落

## 验收标准

- [ ] 回答内容与文档内容一致，不出现明显幻觉
- [ ] 引用角标正确对应检索到的文档片段
- [ ] hover 浮层显示文档名和相关段落
- [ ] 工具调用卡片在对话中正常渲染（显示"已检索"+ 关键词摘要）
- [ ] 流式输出正常，回答逐字显示不卡顿

## 异常场景

- 问题与知识库内容无关时，Agent 明确告知无相关信息，不编造答案
- rag_search 检索失败时，Agent 给出友好提示而非崩溃
