# P0-04 用 /ppt 命令触发 PPT 生成

## 用户故事

作为培训负责人，我希望通过简单的斜杠命令告诉 AI 生成培训 PPT，而不需要手动编写提示词，让启动生成的门槛尽可能低。

## 场景描述

用户在聊天输入框输入 `/ppt` 触发斜杠命令菜单，选择"生成培训PPT"命令后，可附加额外说明（如受众、重点章节），发送后 Agent 开始执行 PPT 生成流程。

## 前置条件

- 工作区已有至少一份就绪文档
- 用户处于聊天面板

## 核心步骤

1. 用户在输入框输入 `/`，弹出斜杠命令菜单
2. 选择"生成培训PPT"命令，输入框显示命令 Pill 标签
3. 用户可在命令后输入补充说明（如"重点突出安全操作规程"）
4. 点击发送，消息以"命令 Pill + 说明文字"形式显示在对话流
5. Agent 接收到带 `/ppt` 前缀的消息，加载 PPT skill 并开始执行

## 验收标准

- [ ] 输入 `/` 后菜单自动出现，列出可用命令
- [ ] 选中命令后输入框变为"Pill + 文本"复合输入态
- [ ] 发送后对话气泡正确渲染命令 Pill 和说明文字
- [ ] Agent 成功识别 `/ppt` 意图并进入生成流程（工具调用卡片开始出现）
- [ ] Backspace 可清除命令 Pill，恢复普通输入态

## 异常场景

- 未上传文档时触发 /ppt，Agent 提示需先上传培训材料
- 命令菜单中无匹配项时，菜单自动收起
