# 瑞士国际风

## 1. Vibe — 整体气质

冷峻、理性、高警觉感。延续瑞士国际主义的设计基因：用严谨的网格秩序承载严肃的安全主题，用单一高饱和强调色触发注意力，用极大的字号对比建立清晰的信息层级。整体不追求装饰堆砌，而是让每一页都像一张经过精密排版的工业海报——克制、清晰、有力。

气质关键词：工业感、网格秩序、单一焦点、理性警示、模块化排版。

## 2. Color System — 色彩系统

以安全橙为视觉锚点，搭配高级灰白与近黑色，形成强对比但不过度喧闹的色盘。强调色只在小面积出现，用于点燃视觉焦点；大面积仍以中性色为主，保证长时间观看的舒适度。

```css
:root {
    --bg-primary: #fafaf8;      /* 主背景：高级灰白 */
    --bg-secondary: #f0f0ee;    /* 次级背景：浅灰卡片/分栏 */
    --text-primary: #0a0a0a;    /* 主文字：近黑 */
    --text-secondary: #525252;  /* 次级文字：中灰 */
    --accent-main: #FF6B35;     /* 主强调色：安全橙 */
    --accent-highlight: #FF8F5C;/* 强调提亮：安全橙浅调 */
    --accent-muted: rgba(255, 107, 53, 0.18); /* 弱化强调：半透明安全橙 */
    --surface: #ffffff;         /* 卡片表面：纯白 */
}
```

| 色彩角色 | 色值 | 使用场景 |
|---|---|---|
| 高级灰白 | `#fafaf8` | 主背景、浅色页面底色 |
| 浅灰 | `#f0f0ee` | 卡片背景、分栏衬底、数据塔顶部图标区 |
| 中灰 | `#525252` / `#737373` | 正文说明、辅助标注、页眉页脚 |
| 近黑 | `#0a0a0a` | 主标题、深色页面背景、主要文字 |
| 安全橙 | `#FF6B35` | 单一强调色：关键数字、标题高亮、重点卡片、分隔线、当前导航点 |
| 安全橙提亮 | `#FF8F5C` | 暗底上的高亮强调、hover 状态 |
| 半透明橙 | `rgba(255,107,53,0.18)` | 弱化强调背景、装饰性几何 |

页面明暗节奏：封面以安全橙满屏开场，正文页在浅底与深底之间交替，形成呼吸感；强调色不连续超过两页，避免视觉疲劳。

## 3. Typography — 字体系统

拒绝系统默认字体，采用几何无衬线搭配等宽字体，营造精密、现代的工业气质。中文标题使用思源黑体或苹方，保证大字重下的清晰可读；西文标题使用 Inter 的极细字重，形成强烈的视觉张力；等宽字体用于页眉、编号、标签，强化技术文档感。

字体来源：全部从中国可访问的 Google Fonts 镜像 `fonts.loli.net` 加载。

| 用途 | 中文字体 | 西文字体 | 字重 | 字号特征 |
|---|---|---|---|---|
| 超大标题 | Noto Sans SC / PingFang SC | Inter | 200–300 | 极大（可达 8–12vw），紧凑行高 |
| 页面主标题 | Noto Sans SC / PingFang SC | Inter | 200–300 | 大标题（5–7vw），负字间距 |
| 小标题/卡片标题 | Noto Sans SC / PingFang SC | Inter | 400–500 | 中等（1.5–2vw） |
| 正文 | Noto Sans SC / PingFang SC | Inter | 400 | 18px 或 1vw 左右，行高 1.5–1.6 |
| 编号/标签/页眉 | Noto Sans SC（辅助） | JetBrains Mono / IBM Plex Mono | 500–600 | 14px，大写，宽字间距 |
| 数据数字 | — | Inter | 200–800 | 随数据重要性从 3vw 到 22vw 不等 |

字重阶梯原则：字号越大，字重越轻；字号越小，字重越重。形成清晰的视觉节奏。

## 4. Layout Grammar — 布局语法

整体基于 12 列模块化网格与 8px 基线模数。页面以全屏卡片容器承载内容，四周保留 5–6vh 的安全边距，底部为分页导航预留约 8vh 安全区。内容不贴边，所有元素严格对齐网格。

页面类型聚合如下：

### Cover / 封面页
- 出现频次：1 次
- 布局结构：全屏单色背景，中央放置超大主标题，顶部为页眉标签，底部为说明文字与翻页提示
- 背景处理：安全橙满屏 + ASCII 字符呼吸场动画
- 核心元素及相对位置：
  - 顶部页眉：靠上分散对齐，左侧为主题标签，右侧为页码
  - 中央主标题：垂直居中偏上，分两行排列，第二行可用斜体强调
  - 底部说明区：左下角为培训说明，右下角为翻页提示
- 占位符：`封面主标题`、`封面副标题`、`页眉主题标签`、`页码`、`培训说明`、`翻页提示`
- 文本颜色规则：主标题纯白，辅助文字为 60–80% 白色透明度

### Content / 内容页
- 出现频次：8 次
- 通用结构：顶部页眉 + 标题区 + 主体内容区 + 底部脚注
- 背景处理：浅底（`#fafaf8`）与深底（`#0a0a0a`）交替，正文卡片用半透明浅色或安全橙突出关键项
- 布局变体（按内容语义选择）：
  1. **KPI 数据塔**：底部对齐的垂直柱塔，顶部小图标，柱体高度对应数值，最高项用安全橙突出
  2. **四列卡片**：等宽四列，每列顶部编号，最后一列用强调色反转
  3. **横向时间线**：页面中央一条水平虚线轴，节点上下交错排列，当前节点用强调色
  4. **三力对比**：左侧大标题卡片，右侧三行编号条目，编号用超大安全橙数字
  5. **六宫格卡片**：3×2 网格，每张卡片含图标、标题、说明，末张用强调色
  6. **双栏对比**：左右两栏，中间竖线分隔，右侧为重点改进项，用强调色
  7. **三列陈述**：底部三列卡片，每列含英文标签、标题、说明、超大编号
  8. **循环图示**：左文右图，右侧为同心圆环与箭头闭环，关键节点用安全橙标记
- 占位符：`页面主标题`、`页面英文副标题`、`页眉左侧主题`、`页眉右侧页码`、`主体内容项 1–N`、`底部脚注`
- 文本颜色规则：浅底用近黑主文+中灰次文；深底用纯白主文+60%白辅助文；强调色用于编号、标签、关键标题

### Closing / 结尾页
- 出现频次：1 次
- 布局结构：左右半屏分栏，左侧为安全橙背景+ASCII呼吸场+宣言，右侧为白底要点列表
- 背景处理：左半屏安全橙，右半屏高级灰白
- 核心元素及相对位置：
  - 左侧：顶部页码、中央宣言大标题、底部署名与日期
  - 右侧：顶部标题、中部三条要点列表、底部结束语
- 占位符：`结尾宣言主标题`、`结尾说明`、`署名信息`、`日期`、`要点 1–3`、`结束语`
- 文本颜色规则：左侧白色/半透明白，右侧近黑主文+中灰次文，最后一项用安全橙强调

## 5. Signature Elements — 标志性元素

1. **ASCII 字符呼吸场**：封面与封底使用全屏字符点阵动画，字符在强调色背景上以低透明度闪烁流动，营造数字化的警觉氛围。
2. **WebGL 漂移网格背景**（可选）：浅色页面使用缓慢漂移的细线网格+微弱点阵；深色页面反转线条颜色。低功耗模式下自动隐藏。
3. **点阵/十字/圆圈矩阵装饰**：使用 CSS 径向渐变或 SVG mask 生成规则排列的几何纹理，作为大面积背景装饰，透明度控制在 0.1–0.5。
4. **极细分隔线**：1–2px 的实线分隔标题区、卡片区、列表项，浅底用近黑线（低透明度），深底用白色线（低透明度）。
5. **单一安全橙色块**：只在最关键的位置出现——当前节点、最高数据柱、重点卡片、最后一项要点，避免多色竞争。
6. **超大编号**：01、02、03 等编号使用 4–6vw 的极细字重，既是信息也是装饰。
7. **等宽标签系统**：页眉、小标签、脚注使用等宽字体+大写+宽字间距，形成技术文档的理性气质。
8. **直角纯色块**：所有卡片、色块、图片容器均为直角，无圆角、无阴影、无渐变填充。

## 6. Visual Assets — 视觉资产

本风格不依赖外部图片资源，所有视觉效果由 CSS 生成。图标使用开源线性图标库（如 Lucide），通过 CDN 加载，统一细线描边风格，颜色跟随当前文字色或安全橙。

若后续需要加入图片：
- 图片容器必须为直角，无圆角、无阴影；
- 图片仅作为信息补充，不得成为视觉主角；
- 暗底页面图片需降低亮度或加半透明遮罩，确保文字可读性；
- 平庸的写实插画建议替换为几何抽象图形或 CSS 渐变背景。

## 7. Usage Guidelines — 生成新 PPT 时的使用说明

1. **主题适配**：优先用于安全、培训、合规、应急、工业等需要理性与警觉气质的主题。非安全类主题可将安全橙替换为其他单一高饱和强调色（如克莱因蓝、柠檬黄），但需保持“单一强调色+中性底”的原则。
2. **页数建议**：8–12 页为宜，保持每页一个核心观点。页面类型顺序建议：封面 → 数据 → 概念 → 流程 → 原则 → 操作 → 对比 → 责任 → 闭环 → 结尾。
3. **明暗节奏**：避免连续三页同底色，建议在浅底与深底之间交替，封面与结尾可用强调色打破节奏。
4. **字体加载**：必须从中国可访问的 `fonts.loli.net` 加载字体，中文优先 Noto Sans SC，西文优先 Inter + JetBrains Mono。
5. **网格约束**：所有内容对齐 12 列网格，边距、间距优先使用 8px 模数（8/16/24/32/48/64/80/96）。
6. **强调色使用**：每页最多 1–2 处使用安全橙，用于最重要的数据或结论，避免滥用。
7. **动效语义**：动画应服务于内容表达——数据塔从底部生长、时间线节点依次点亮、对比两栏从两侧滑入。避免无意义的弹跳或过度装饰。
