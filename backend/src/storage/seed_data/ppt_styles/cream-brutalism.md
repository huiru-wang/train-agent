# 奶油粗野主义

## 1. Vibe — 整体气质

明快、利落、有编辑感。以接近纸张的奶白色为底，以粗黑边框建立强烈的结构骨架，再以薄荷绿、浅蓝、柔粉、柠黄等糖果色块作为信息分区。整体气质介于「独立信息图表」与「趋势解读长图」之间——既用大号数字与标题制造视觉冲击力，又用彩色卡片与标签让复杂信息一眼可扫。

关键词：粗黑边框 / 糖果色块 / 大号数据 / 杂志标签 / 强结构

## 2. Color System — 色彩系统

```css
:root {
    --bg-primary: #FAF9F6;
    --bg-secondary: #FFFFFF;
    --text-primary: #111111;
    --text-secondary: #555555;
    --accent-main: #111111;
    --accent-highlight: #A8E6A3;
    --accent-muted: #B8E7F7;
    --surface: #F0F0F0;
    --accent-pink: #F9B1E0;
    --accent-yellow: #FFD66B;
    --accent-cream: #F5F1E8;
}
```

| 变量 | 色值 | 角色 | 使用场景 |
|------|------|------|----------|
| `--bg-primary` | #FAF9F6 | 奶白纸 | 页面主背景、深色文字 |
| `--bg-secondary` | #FFFFFF | 纯白 | 卡片底色、引用框背景 |
| `--text-primary` | #111111 | 墨黑 | 主标题、正文、数字、边框 |
| `--text-secondary` | #555555 | 中灰 | 辅助说明、来源标注、弱化文字 |
| `--accent-main` | #111111 | 结构黑 | 外边框、卡片边框、页眉页脚文字 |
| `--accent-highlight` | #A8E6A3 | 薄荷绿 | 重点卡片、高亮标签、正向信息 |
| `--accent-muted` | #B8E7F7 | 浅蓝 | 次要卡片、说明标签、引用条 |
| `--surface` | #F0F0F0 | 卡片表面 | 中性信息块、底部解说条 |
| `--accent-pink` | #F9B1E0 | 柔粉 | 强调卡片、结论标签、引人注意的信息 |
| `--accent-yellow` | #FFD66B | 柠黄 | 数据高亮、年限标签、关键数字 |
| `--accent-cream` | #F5F1E8 | 米灰 | 对比卡片中的「旧/传统」侧背景 |

**配色逻辑**：每份 deck 以奶白 + 墨黑为不变骨架，彩色卡片根据信息角色分配：绿色用于正向/定义/方案，蓝色用于说明/来源/过程，粉色用于结论/强调/争议，黄色用于数据/年限/高亮。彩色块只作为卡片填充色或标签底色使用，不用于大标题或正文。

## 3. Typography — 字体系统

| 用途 | 中文字体 | 西文字体 | 字重 |
|------|----------|----------|------|
| 超大标题 / 金句 | Noto Sans SC | Archivo Black | Black / 900 |
| 页面主标题 | Noto Sans SC | Archivo | Bold / 700 |
| 副标题 / 引导段 | Noto Sans SC | Space Grotesk | Medium / 500 |
| 正文 / 说明 | Noto Sans SC | Nunito | Regular / 400 |
| 页眉 / 页脚 / 标签 | Noto Sans SC | Space Grotesk | Medium / 500 |
| 数字 / KPI | Noto Sans SC | Archivo | Bold / 700 |

**导入示例**：

```
https://fonts.loli.net/css2?family=Archivo+Black&family=Archivo:wght@400;500;600;700&family=Space+Grotesk:wght@400;500;600;700&family=Nunito:wght@400;500;600;700&family=Noto+Sans+SC:wght@400;500;700;900&display=swap
```

**排版规则**：
- 页面主标题：几何无衬线 Black，视口宽度 6–10vw，左对齐为主，字间距 -0.02em
- 英文副标题 / 引导小字：Space Grotesk 全大写，12–14px，字间距 +0.12em，颜色取 `--text-secondary`
- 正文：Noto Sans SC / Nunito，15–18px，行高 1.5–1.65
- 数字：Archivo Bold，用于数据页与 KPI，字间距 -0.02em
- 卡片标题：Noto Sans SC Bold，20–28px
- 标签文字：Space Grotesk Medium，全大写或首字母大写，11–13px

**核心原则**：几何无衬线负责「视觉冲击」（标题、数字、标签），人文无衬线负责「可读信息」（正文、说明），所有文字保持高对比，禁止在彩色卡片上使用彩色文字。

## 4. Layout Grammar — 布局语法

### Cover / 封面页

出现频次：1 页

**结构**：页面由粗黑外框包围。顶部左侧为圆角药丸形栏目 logo，顶部右侧为横向主题标签。主体左侧垂直排列：黑色引导标签 → 超大英文缩写标题 → 彩色底黑色文字主标题 → 副标题/引导段。右侧为 3 个纵向堆叠的彩色条目卡片，每个卡片包含条目编号与条目标题。底部左侧为说明文字，底部中央或右侧为底部解说条或比例标注。

**背景**：`--bg-primary` 纯色，可加极淡的噪点纹理或网格点阵作为装饰层。

**占位符**：`栏目 logo`、`顶部主题标签`、`引导标签`、`英文大标题`、`中文主标题`、`副标题`、`右侧条目 1 标题`、`右侧条目 2 标题`、`右侧条目 3 标题`、`底部说明`、`比例标注`

**文字颜色**：主标题与英文标题用 `--text-primary`，彩色卡片内文字用 `--text-primary`，底部说明用 `--text-secondary`。

---

### Data / 数据大字报页

出现频次：1–2 页

**结构**：顶部为药丸形引导标签。主体左侧为超大数字/KPI，右侧为中文标题与来源说明。底部保留细长进度条或分隔线，下方为页脚说明。数字占据页面视觉中心，标题与数字形成左右呼应。

**背景**：`--bg-primary` 纯色。

**占位符**：`引导标签`、`核心数字`、`数字单位`、`数据标题`、`数据来源`、`页脚说明`

**文字颜色**：数字与标题用 `--text-primary`，来源与页脚用 `--text-secondary`。

---

### Content / 内容页 — 双栏对比型

出现频次：1 页

**结构**：顶部为药丸形引导标签。主体为左右 1:1 双栏。左栏为浅色/米灰底卡片，代表「旧/传统/ Before」，内部为标签、标题与要点；右栏为彩色底卡片（薄荷绿），代表「新/方案/ After」，内部为标签、标题与要点。两栏之间用粗黑箭头连接。底部可附加一行解说条。

**背景**：`--bg-primary` 纯色。

**占位符**：`引导标签`、`左栏标签`、`左栏标题`、`左栏要点`、`右栏标签`、`右栏标题`、`右栏要点`、`底部解说`

**文字颜色**：两栏内部文字均用 `--text-primary`，左栏可适当降低透明度以形成弱化。

---

### Content / 内容页 — 横向分段条型

出现频次：1 页

**结构**：顶部为药丸形引导标签。主体为一个横向贯穿的彩色分段条，每段标注百分比与文字说明。分段条下方为二级标题，再下方为 3–5 个横向排列的药丸标签或方框标签。最下方为来源标注与底部解说条。

**背景**：`--bg-primary` 纯色。

**占位符**：`引导标签`、`分段条标题 1`、`分段条占比 1`、`二级标题`、`标签 1`（共 3–5 组）、`来源标注`、`底部解说`

**文字颜色**：分段条内文字用 `--text-primary`，标签文字用 `--text-primary`，来源与解说用 `--text-secondary`。

---

### Content / 内容页 — 三栏卡片型

出现频次：1–2 页

**结构**：顶部为页面主标题（可带副标题）。主体为横向排列的三张等宽彩色卡片，每张卡片左上角为编号，中部为卡片标题，下部为说明要点。卡片之间保持等距间隙。底部可附加解说条。

**背景**：`--bg-primary` 纯色。

**占位符**：`页面主标题`、`页面副标题`、`卡片 1 编号`、`卡片 1 标题`、`卡片 1 说明`、`卡片 2 编号`、`卡片 2 标题`、`卡片 2 说明`、`卡片 3 编号`、`卡片 3 标题`、`卡片 3 说明`、`底部解说`

**文字颜色**：卡片标题用 `--text-primary`，说明文字用 `--text-secondary`，编号用 `--text-primary`。

---

### Content / 内容页 — 双栏信息卡型

出现频次：0–1 页

**结构**：顶部为药丸形引导标签。主体为左右两个彩色信息卡片，每个卡片顶部为深色标题条（内含图标或来源名），下方为职责、要求、技术等结构化信息。底部可附加解说条。

**背景**：`--bg-primary` 纯色。

**占位符**：`引导标签`、`左卡标题条`、`左卡职责`、`左卡要求`、`左卡技术`、`右卡标题条`、`右卡职责`、`右卡要求`、`右卡技术`、`底部解说`

**文字颜色**：标题条文字用 `--bg-primary`，卡片内文字用 `--text-primary`。

---

### Closing / 结尾页

出现频次：1 页

**结构**：与封面页类似，由粗黑外框包围。中央放置大号结语标题，下方为行动号召或总结说明。底部保留页脚说明与比例标注。

**背景**：`--bg-primary` 纯色，或采用深色反色版本（`--text-primary` 作为背景，`--bg-primary` 作为文字）。

**占位符**：`结语标题`、`总结说明`、`页脚说明`、`比例标注`

**文字颜色**：浅色背景用 `--text-primary`，深色反色版用 `--bg-primary`。

## 5. Signature Elements — 标志性元素

### 粗黑外框与卡片边框

每页最外层必须有一圈 3–4px 的 `--text-primary` 实线边框，与奶白背景形成强烈的粗野主义边界。内部所有信息卡片、标签、按钮也使用 2–3px 的同款黑色边框，保持整体语言一致。

### 彩色底黑字卡片

信息卡片统一采用彩色填充（绿/蓝/粉/黄）+ 黑色边框 + 黑色文字。禁止在彩色卡片上使用白色文字或渐变填充。彩色卡片应带有轻微的错位黑色阴影（offset shadow），增强拼贴感。

### 药丸形标签

页面顶部或标题上方使用圆角药丸形标签，内部为引导主题或章节名。标签底色可为彩色或白色，但必须带有黑色边框。标签文字使用几何无衬线，全大写或首字母大写，字间距略宽。

### 错位阴影

主要卡片和标签下方带有向右下方偏移的黑色实色阴影（不使用模糊阴影），形成类似剪纸拼贴的立体层次。阴影偏移量统一为 4–6px。

### 底部解说条

每页底部可放置一条半透明的深灰或黑色细长条，用于承载演讲者备注、补充说明或转述文字。该条位于页面内容区下方、页脚上方，文字使用 `--bg-primary` 或浅灰色。

### 顶部页眉条

每页顶部固定：左侧为圆角药丸形栏目 logo，右侧为当前主题或章节名。两者均使用 Space Grotesk，字间距略宽。

### 页脚比例标注

每页右下角固定标注「横版 · 16:9」或类似比例信息，使用小字号灰色文字，作为版式签名。

## 6. Visual Assets — 视觉资产

本风格不依赖外部图片资源，所有视觉效果由 CSS 生成。

图标统一使用简洁线性图标库，**禁止使用 emoji**。常用图标方向：公司/来源标识、箭头、数据、标签、流程。

若需要使用来源 logo，应使用简化的首字母或文字标识替代真实品牌 logo，保持风格统一。

## 7. Usage Guidelines — 生成新 PPT 时的使用说明

### 整体原则
1. **一套骨架四色走到底**：奶白背景 + 墨黑边框贯穿全 deck，糖果色只用于卡片/标签，不替换骨架色。
2. **粗边框是语言**：所有结构性元素必须有 2px 以上黑色实线边框，禁止细线或无框设计。
3. **彩色卡片承载信息角色**：绿色=定义/方案/正向，蓝色=说明/来源/过程，粉色=结论/强调/争议，黄色=数据/年限/高亮。
4. **大号数字优先**：数据页让数字成为主角，标题与来源作为配角，避免数字过小或信息过载。
5. **不使用 emoji**：所有图标意义用线性图标或文字标签表达。

### 页面生成规则
- **封面页**：奶白底，粗黑外框，左上药丸 logo，右上主题标签，左侧大标题，右侧 3 个纵向彩色条目卡片。
- **数据大字报页**：奶白底，顶部药丸标签，左侧超大数字，右侧标题与来源，底部页脚说明。
- **双栏对比页**：左右 1:1 卡片，左侧米灰（旧），右侧薄荷绿（新），中间粗黑箭头，底部解说条。
- **横向分段条页**：顶部标签，横向彩色分段条，下方二级标题 + 药丸标签阵列，底部来源与解说。
- **三栏卡片页**：顶部主标题，下方 3 张等宽彩色卡片，每张含编号、标题、说明。
- **双栏信息卡页**：顶部标签，左右两张信息卡，顶部深色标题条，下方结构化信息。
- **结尾页**：奶白底或深色反色，中央大号结语标题，底部页脚说明。

### 动效建议
- 封面页：元素从屏幕外 stagger 滑入，营造拼贴展开感。
- 数据页：数字采用计数动画，标题与来源随后淡入。
- 对比页：左右两栏分别滑入，箭头最后出现。
- 分段条页：各分段依次延展填充。
- 三栏卡片页：三张卡片从左到右依次弹出。
- 其他正文页：采用级联淡入 + 轻微上移即可。

### 禁止事项
- 禁止自定义骨架颜色（奶白底/墨黑框不可替换）。
- 禁止在彩色卡片上使用白色或浅色文字。
- 禁止使用 emoji。
- 禁止使用真实摄影图片作为页面背景。
- 禁止使用模糊阴影或玻璃拟态效果。
- 禁止让页面元素紧贴外框，必须保留 3–5% 的内边距。
