# 暗色柔光

## 1. Vibe — 整体气质

优雅、克制、温暖而高级。以近乎纯黑的深底为画布，通过柔焦的暖色光斑与纤细的几何线条营造艺术感。整体不依赖写实插图，完全由抽象光影、衬线字体和留白的呼吸感构成，适合需要强调品质感与情绪共鸣的宣讲、发布会或品牌叙事场景。

关键词：暗色沉浸 / 柔焦光晕 / 衬线强调 / 暖色点缀 / 极简几何

## 2. Color System — 色彩系统

```css
:root {
    --bg-primary: #0f0f0f;
    --bg-secondary: #1a1a1a;
    --text-primary: #e8e4df;
    --text-secondary: #9a9590;
    --accent-main: #d4a574;
    --accent-highlight: #e8b4b8;
    --accent-muted: #c9b896;
    --surface: rgba(232, 228, 223, 0.08);
}
```

| 变量 | 色值 | 角色 | 使用场景 |
|------|------|------|----------|
| `--bg-primary` | #0f0f0f | 深黑底 | 所有页面主背景 |
| `--bg-secondary` | #1a1a1a | 卡片底 | 卡片、内容容器背景 |
| `--text-primary` | #e8e4df | 主文字 | 标题、核心文案 |
| `--text-secondary` | #9a9590 | 辅助文字 | 正文、说明、弱化信息 |
| `--accent-main` | #d4a574 | 暖金强调 | 关键词、标签边框、按钮、装饰线 |
| `--accent-highlight` | #e8b4b8 | 柔粉高亮 | 柔焦光晕、次要高亮、情感点缀 |
| `--accent-muted` | #c9b896 | 香槟金 | 引言页主色、奖项金额、金色质感元素 |
| `--surface` | rgba(232,228,223,0.08) | 微妙表面 | 卡片边框、分割线、hover 底色 |

**配色逻辑**：以深黑为绝对主调，暖金、柔粉、香槟金仅作为氛围与高亮点缀。强调信息通过衬线字重、斜体与光晕位置完成，不依赖大面积彩色块。

## 3. Typography — 字体系统

| 用途 | 中文字体 | 西文字体 | 字重 / 样式 |
|------|----------|----------|-------------|
| 封面主标题 | Noto Serif SC | Cormorant | 600 / 常规或斜体 |
| 页面主标题 / 金句 | Noto Serif SC | Cormorant | 400 / 可斜体 |
| 卡片标题 | Noto Serif SC | Cormorant | 600 / 可斜体 |
| 正文 / 说明 | Noto Sans SC | IBM Plex Sans | 300 / 400 |
| 标签 / 按钮 | Noto Sans SC | IBM Plex Sans | 400 / 500 |

**导入示例**：

```
https://fonts.loli.net/css2?family=Cormorant:ital,wght@0,400;0,600;1,400;1,600&family=IBM+Plex+Sans:wght@300;400;500&family=Noto+Serif+SC:wght@400;600;700&family=Noto+Sans+SC:wght@300;400;500&display=swap
```

**排版规则**：
- 封面主标题：衬线，视口宽度 5–8vw，居中，关键词用斜体暖色突出
- 页面主标题：衬线，视口宽度 3–4vw，居中或顶部居中
- 引言引用：衬线斜体，视口宽度 4–6vw，居中，香槟金色
- 正文：无衬线，14–18px，行高 1.6，字重 300
- 标签 / 按钮：无衬线，小字号，暖色边框或填充

**核心原则**：衬线用于「视觉重音」（标题、金句、卡片标题），无衬线用于「信息密度」（正文、说明、标签）。

## 4. Layout Grammar — 布局语法

### Cover / 封面页

出现频次：1 页

**结构**：全屏居中布局。顶部为抽象的柔焦光晕与细垂直装饰线，中央垂直居中排列：英文引导小字 → 超大主标题 → 一行说明。

**背景**：`--bg-primary` 纯色深底，右上角、左下角、中心叠加柔焦暖色光晕，左右两侧各一条细垂直渐变线。

**占位符**：`封面英文副标题`、`封面主标题`、`封面说明`

**文字颜色**：主标题用 `--text-primary`，副标题与关键词用 `--accent-main`，说明用 `--text-secondary`。

---

### Statement / 金句页

出现频次：0–1 页

**结构**：全屏居中，中央垂直排列：大号斜体引用 → 一行补充说明。引用可分 2 行断行以制造节奏。

**背景**：`--bg-primary` 深底，对角分布柔焦光晕。

**占位符**：`引用文字`、`引用说明`

**文字颜色**：引用用 `--accent-muted` 衬线斜体，说明用 `--text-secondary`。

---

### Content / 内容页 — 三列卡片型

出现频次：1 页

**结构**：顶部居中页面主标题 → 下方三等分网格卡片。每张卡片结构固定：顶部图标 → 卡片标题 → 说明文字 → 底部圆角标签。

**背景**：`--bg-primary` 深底，对角柔焦光晕。

**占位符**：`页面主标题`、`卡片 1 图标`、`卡片 1 标题`、`卡片 1 说明`、`卡片 1 标签`（共 3 组）

**文字颜色**：卡片标题用 `--accent-main`，说明用 `--text-secondary`，标签文字与边框用 `--accent-main`。

---

### Content / 内容页 — 数据卡片型

出现频次：0–1 页

**结构**：顶部居中页面主标题 → 下方三等分卡片网格，中间卡片可 featured 放大。每张卡片结构：顶部图标 → 名次 → 巨型金额数字 → 附加说明。

**背景**：`--bg-primary` 深底，对角柔焦光晕。

**占位符**：`页面主标题`、`奖项卡片 1 图标`、`奖项卡片 1 名次`、`奖项卡片 1 金额`、`奖项卡片 1 说明`（共 3 组）

**文字颜色**：普通卡片金额用 `--accent-muted`，featured 卡片金额用 `--accent-main`，名次与说明用 `--text-secondary`。

---

### Content / 内容页 — 流程步骤型

出现频次：0–1 页

**结构**：顶部居中页面主标题 → 下方垂直编号列表。每条步骤结构：左侧圆形编号 → 右侧步骤标题 → 步骤说明。条目之间使用 subtle 分割线。

**背景**：`--bg-primary` 深底，柔焦光晕。

**占位符**：`页面主标题`、`步骤 1 编号`、`步骤 1 标题`、`步骤 1 说明`（共 3–5 组）

**文字颜色**：编号圆环与数字用 `--accent-main`，标题用 `--text-primary`，说明用 `--text-secondary`。

---

### Closing / 结尾页

出现频次：1 页

**结构**：全屏居中布局。中央垂直排列：大号结语标题 → 一行补充说明 → 圆角行动按钮。标题关键词用斜体暖色突出。

**背景**：`--bg-primary` 深底，柔焦光晕 + 左右垂直装饰线。

**占位符**：`结尾主标题`、`结尾说明`、`行动按钮`

**文字颜色**：主标题用 `--text-primary`，强调词用 `--accent-main`，说明用 `--text-secondary`，按钮文字用深色 `#0f0f0f`。

## 5. Signature Elements — 标志性元素

### 柔焦渐变光晕

使用高斯模糊的径向渐变圆，颜色为暖金、柔粉、香槟金，透明度约 0.3–0.4，分布于页面四角或中心作为氛围背景。每页保持 2–3 个，避免铺满画面。

### 细垂直装饰线

页面左右两侧各一条 1px 宽、由上至下渐隐的暖色竖线，提升仪式感与纵向节奏。主要用于封面页与结尾页。

### 斜体衬线强调

标题中的关键词使用 Cormorant / Noto Serif SC 斜体，并以 `--accent-main` 暖金色突出，形成签名式的视觉记忆点。

### 圆角深色卡片

内容页使用 16px 圆角、深灰半透明卡片（`--bg-secondary`），配合 `--surface` subtle 边框。hover 时边框可微亮为 `--accent-main` 20% 透明度。

### 圆形编号徽章

步骤列表中的序号使用描边圆形，内部为 `--accent-main` 数字，直径约 32–44px。

### 圆角胶囊按钮

CTA 使用 `--accent-main` 填充的圆角胶囊按钮，文字为深色 `#0f0f0f`，padding 适中以形成清晰的行动召唤。

### 无写实插图

所有视觉效果由 CSS 生成，不使用摄影图、通用插画或复杂图标。如需图标，使用极简符号或 emoji，并保持低饱和度处理。

## 6. Visual Assets — 视觉资产

本风格不依赖外部图片资源，所有视觉效果由 CSS 生成。若需使用图标，建议使用极简符号、emoji 或几何抽象图形，并保持低饱和度或单色调处理，避免破坏暗色柔光的整体氛围。

## 7. Usage Guidelines — 生成新 PPT 时的使用说明

1. **页面类型选择**：优先用于 Cover、Statement、三列卡片、数据卡片、流程步骤、Closing 六种页面；避免信息密度过高的复杂表格页。
2. **标题处理**：主标题宜短，关键词用斜体衬线突出；避免超过两行的大标题。
3. **配色保持**：始终维持深底浅色字，强调色仅用于 10%–20% 的关键信息，避免大面积暖色块。
4. **光晕分布**：每页保持 2–3 个柔焦圆即可，featured 卡片页可让暖色光晕靠近重点卡片。
5. **卡片数量**：三列网格页建议 3–6 个卡片，保持列数不超过 3；步骤列表页建议 3–5 步。
6. **字体禁忌**：标题禁用 Calibri / Arial / Microsoft YaHei，正文保持 IBM Plex Sans / Noto Sans SC 的轻盈感。
7. **留白优先**：居中页面保持充足内边距，内容区宽度不超过 1000px，确保大屏幕下依然聚焦。
8. **按钮使用**：每份 PPT 的结尾页可放置一个 CTA 按钮，文案简洁有力，颜色使用 `--accent-main`。
