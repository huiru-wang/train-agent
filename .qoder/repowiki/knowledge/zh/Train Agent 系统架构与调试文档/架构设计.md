- **双进程后端架构**：FastAPI (:8000) 负责 REST CRUD 与文件处理，LangGraph Server (:2024) 负责 Agent 流式推理，两者共享 SQLite/ChromaDB/FileStore 存储层。
- **四层后端设计**：自上而下分为 API 层（路由/依赖注入）、Agent 层（ReAct 图/中间件/技能管理）、Service 层（文档解析流水线编排）和 Storage 层（异步 SQLite/向量索引/文件系统）。
- **前端三栏布局**：Next.js App Router 驱动，左侧文档面板（上传/状态轮询）、中间聊天面板（LangGraph Stream 通信/表单中断）、右侧产出面板（预览/下载）。
- **通信分离策略**：REST API 处理元数据与工作区管理，@langchain/react useStream Hook 处理实时对话与工具调用反馈。
