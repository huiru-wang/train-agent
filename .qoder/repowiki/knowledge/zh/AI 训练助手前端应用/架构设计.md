- **框架与路由**：采用 Next.js (App Router) 架构，通过 `src/app` 实现文件系统路由（如 `/workspace/[id]`）。
- **组件分层**：遵循 `lib` (API/工具) -> `components` (UI/业务逻辑) -> `app` (页面布局) 的分层结构。
- **状态管理**：使用 React Context (`StreamContext`, `ResumeContext`) 封装 LangGraph 流式对话状态，结合 `zustand` 进行全局状态管理。
- **布局模式**：核心工作区采用自定义的 `ThreePanel` 三栏布局（文档、对话、任务），支持动态拖拽调整宽度。
- **数据交互**：在 `src/lib/api.ts` 中统一封装 RESTful API 请求，集成 LangChain/LangGraph SDK 处理流式 AI 响应。
