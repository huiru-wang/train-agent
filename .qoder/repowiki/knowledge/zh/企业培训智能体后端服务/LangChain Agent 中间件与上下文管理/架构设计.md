- **入口与导出**：`__init__.py` 统一导出核心中间件工厂函数（如 `create_inject_doc_context`）和装饰器实例（如 `log_before_agent`），作为模块对外接口。
- **动态提示词注入**：`inject_doc_context.py` 利用 `@dynamic_prompt` 装饰器，在模型请求前从数据库检索工作区文档并动态拼接至系统提示词中。
- **全链路日志监控**：`logging_middlewares.py` 通过 `@before_agent`、`@after_model` 等装饰器覆盖 Agent 执行的关键节点，记录工作区 ID、消息数量及工具调用详情。
- **消息兼容性处理**：`model_message_sanitizer.py` 使用 `@wrap_model_call` 拦截模型请求，清洗不支持的消息类型（如无效的工具调用），确保与 OpenAI 兼容 API 的交互稳定性。
- **智能对话摘要**：`summarization.py` 继承 `SummarizationMiddleware`，实现基于消息数量和令牌的冷却机制，并通过自定义标记（`train_agent_hidden`）隐藏 UI 中的摘要信息。
