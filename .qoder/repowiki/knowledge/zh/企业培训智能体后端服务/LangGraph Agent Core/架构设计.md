- **入口与编排**：`graph.py` 是核心入口，通过 `create_graph` 工厂函数组装 `ChatOpenAI` 模型、工具集（RAG、技能加载等）及中间件链。
- **状态管理**：`state.py` 定义 `TrainAgentState`，继承自 `AgentState` 并扩展 `workspace_id` 字段以支持多工作区上下文。
- **技能系统**：`skill_manager.py` 实现基于文件系统（SKILL.md + YAML Frontmatter）的技能发现与按需加载机制，遵循渐进式披露原则。
- **中间件体系**：采用 LangChain Middleware 模式，包含 `MessageHistoryMiddleware`（异步持久化聊天记录至数据库）、`TrainAgentSummarizationMiddleware`（基于 Token 阈值和消息冷却期的对话摘要）以及日志/安全过滤中间件。
- **提示词工程**：`prompt_manager.py` 集中管理系统级提示词，规范了引用格式（{{ref:...}}）与 Markdown 输出标准。
