- **分层架构设计**：采用前后端分离架构，前端为 Next.js + assistant-ui，后端为 LangGraph Python Server，通过 SSE 实现流式对话，REST API 处理资源管理。
- **Agent 核心组件**：内部划分为 Context Manager（四层上下文策略）、Tool Manager（RAG/表单/技能加载）和 Skill Manager（基于 SKILL.md 的零代码扩展机制）。
- **数据与存储模型**：定义了 SQLite 结构化存储（Workspace/Document/Task）与 ChromaDB 向量存储（按 Workspace 隔离 Collection）的协同模式。
- **交互流程规范**：通过 Mermaid 序列图详细描述了从工作区创建、文档向量化处理到多轮 RAG 对话及 PPT 技能执行的全链路状态流转。
